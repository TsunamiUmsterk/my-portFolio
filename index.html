About Me

These are some Games I have made

These games that I have made use p5.js, p5.play.js and matter.js library on two different platforms; Github and p5.js Editor. These four games are some examples of what I have made through coding. The rest of these repositories/ submissions are shown on this page

My T-rex Game

My t-rex game is a replica of the original game that appears on google chrome when the internet isn't working. I made this from scratch last year. It was submitted on 9th January 2021 on Github

My Fruit Catcher Game

My fruit Catcher game is all about trying to beat your rival trying to catch the falling fruits and you can always choose when to start or finish the game. It was submitted on 12th August 2021 on Github

My Ghost Runner Game

My Ghost Runner game is all about trying to stop the ghost from certain doom by repeatedly pressing the space bar and finding safe haven or it's Game Over! It was saved on p5.js Editor

My Angry Birds Game

Very much like the original, my Angry Birds game plays well for a solo player. It was submitted on 8th May 2021 on Github
